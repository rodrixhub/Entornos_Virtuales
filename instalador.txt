Instalación del Proyecto React

Este documento detalla los pasos para instalar y configurar un proyecto en React y Node.js con una base de datos en MongoDB. Asegúrate de tener todas las herramientas necesarias instaladas antes de comenzar.

Requisitos Previos:
1. Node.js y npm: Asegúrate de tener la versión más reciente de Node.js y npm instaladas.
   - Puedes descargar Node.js desde: https://nodejs.org/
   - npm se instala automáticamente con Node.js.

2. Git: Asegúrate de tener Git instalado para clonar el repositorio.
   - Puedes descargar Git desde: https://git-scm.com/

3. MongoDB: Asegúrate de tener MongoDB instalado y en ejecución.
   - Puedes descargar MongoDB desde: https://www.mongodb.com/try/download/community
   - Alternativamente, puedes usar MongoDB Atlas para una base de datos en la nube: https://www.mongodb.com/cloud/atlas

Pasos de Instalación:

1. Clonar los repositorios del proyecto:
   Abre una terminal y ejecuta los siguientes comandos:
   ```sh
   git clone https://github.com/rodrixhub/Entornos_Virtuales.git    # Front
   git clone https://github.com/kruv15/BackEVA.git                  # Back

2. Navegar al directorio donde está ubicado cada proyecto:
   cd Entornos_Virtuales          # Frontend
   cd BackEVA                     # Backend
3. Instalar las dependencias del proyecto:
   npm install                    # Ejecutar en ambos directorios (frontend y backend)
4. Configurar la base de datos MongoDB:
   Si estás utilizando MongoDB localmente, asegúrate de que MongoDB esté en ejecución.
   Si estás utilizando MongoDB Atlas, obtén la URL de conexión de tu clúster.
