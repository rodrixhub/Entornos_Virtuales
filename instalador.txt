Instalación del Proyecto React Grupo2 

Este documento detalla los pasos para instalar y configurar un proyecto en React y Node.js
Requisitos Previos: 
Debes tener instalado react y node.js, tambien git
1. Node.js y npm: Asegúrate de tener la versión más reciente de Node.js y npm instaladas.
   - Puedes descargar Node.js desde: https://nodejs.org/
   - npm se instala automáticamente con Node.js.

2. Git: Asegúrate de tener Git instalado para clonar el repositorio.
   - Puedes descargar Git desde: https://git-scm.com/

Clonación del Repositorio:  
Abre una terminal o línea de comandos.
Navega hasta el directorio donde quieras clonar el proyecto.
1. Ejecuta el siguiente comando para clonar el repositorio:
   git clone https://github.com/rodrixhub/Entornos_Virtuales.git    # Front y Back
(ojo metimos el backend en el mismo repositorio la carpeta se llama BackEva y debes instalar las dependencias en las 2 partes frontend y backend)
2. Navegar al directorio donde está ubicado cada proyecto:
   cd Entornos_Virtuales          # Frontend
   cd BackEVA                     # Backend
3. Instalar las dependencias del proyecto:
   npm install                    # Ejecutar en ambos directorios (frontend y backend)
(con las dependencias listas ya puedes ejecutar el programa)
4. Hacer correr Front y Backend (abrir una terminar en cada carpeta del programa)
npm run dev          # Frontend
npm run start        # Backend
Con eso listo ya puedes hacer correr el programa

------------------------- Manual de uso ----------------------------------
1. iniciar sesion o crearse una cuenta
2. Para ver el video de prueba en la pagina solo hace basta un click y las preguntas asociadas a los minutos apareceran
3. Para poder añadir tu contenido debes dirigirte a la parte superior derecha en el icono del usuario
4. En el icono de usuario se mostrara la parte de la biblioteca y subir video
5. para subir video hacer click en la parte de subir video y podras subir el contenido que necesites
6. para poder ver donde esta tus videos, ve a la parte de la "Biblioteca" ahi estan todos los videos que subiste
7. para añadir preguntas a tu video tienes que estar dentro de "Biblioteca" y hacer click a el video que elejias
8. una ves dentro del video elegido tienes las opciones de añadir pregunta y exportar en formato scorm
9. Para añadir la pregunta debes encontrarte en el minuto que quieras que aparezca la pregunta y añadirla
10 Para exportar en un formato scorm solo basta con hacer click en el boton de exportar a scorm y con eso puedes subirlo a moodle o scorm cloud
